## Installation

### 1. Clone the repository

```bash
git clone https://github.com/amitabhanmolpain/FakeReviewDetector.git
cd FakeReviewDetector
```

### 2. Create & activate a virtual environment

**Linux / macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows PowerShell:**
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

### 3. Install dependencies

```bash
pip install -r backend/requirements.txt
```

### 4. Download NLTK data (one‑time)

```bash
python -c "import nltk; nltk.download('stopwords'); nltk.download('punkt'); nltk.download('punkt_tab')"
```

---

## Train the model

```bash
# First‑time training (skips if models/fake_review_pipeline.pkl already exists)
python backend/train.py

# Force re‑train (overwrites existing pipeline)
python backend/train.py --force
```

## Evaluate the model

```bash
python backend/evaluate.py
```

This prints classification report, confusion matrix, ROC AUC, and saves results to `models/metrics.json`.

---

## Run the API server

**Linux / macOS:**
```bash
uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

**Windows PowerShell:**
```powershell
uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

The API docs are available at [http://localhost:8000/docs](http://localhost:8000/docs).

## Unit Tests (sketch)

```python
# tests/test_predict.py
from fastapi.testclient import TestClient
from backend.main import app

client = TestClient(app)

def test_predict_returns_label():
    resp = client.post("/predict", json={"text": "Amazing product, best purchase ever!!!"})
    assert resp.status_code == 200
    data = resp.json()
    assert data["label"] in ("Fake", "Real", "Uncertain")
    assert 0 <= data["fake_prob"] <= 1
    assert 0 <= data["combined_fake_score"] <= 1

def test_predict_empty_text():
    resp = client.post("/predict", json={"text": ""})
    assert resp.status_code == 422  # validation error

def test_predict_missing_field():
    resp = client.post("/predict", json={"review": "hello"})
    assert resp.status_code == 422
```

Run with: `pytest tests/`
